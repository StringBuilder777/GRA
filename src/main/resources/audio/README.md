# RECURSOS DE AUDIO

Este directorio debe contener:

1. **golpe.wav** - Sonido de golpe para cuando se hace clic en la piñata
   - Puedes descargar sonidos gratuitos de sitios como:
     - https://freesound.org/
     - https://www.zapsplat.com/
   - Busca: "hit", "punch", "impact"

2. **musica.mp3** - Música de fondo navideña
   - Música libre de derechos de sitios como:
     - https://incompetech.com/
     - https://www.bensound.com/
   - Busca música navideña o festiva

## Instalación de MPV (requerido para música de fondo)

### macOS
```bash
brew install mpv
```

### Linux (Ubuntu/Debian)
```bash
sudo apt-get install mpv
```

### Windows
Descarga desde: https://mpv.io/installation/

## Notas
- El sonido de golpe (golpe.wav) usa JavaFX Media
- La música de fondo usa MPV externo para mejor rendimiento
